# Portfolio Laura Reyes

A personal portfolio built with Angular that combines clean architecture, thoughtful UI design and modern frontend practices.

Designed to showcase not only my projects, but also how I approach software development.

## ✨ Features
- 🌅 Sunset / 🌙 Night themes
- 🌎 English / Spanish
- 📱 Fully responsive
- 🎨 Custom Design System
- ♿ Accessibility-first
- ⚡ Modern Angular (Signals, Standalone Components)
- 🚀 Optimized performance
- 🎭 Smooth micro-interactions

## 🧠 Stack
- Angular 20
- TypeScript
- SCSS
- HTML5
- Signals
- Standalone Components
- CSS Variables
- Flexbox
- CSS Grid

## 🏛️ Frontend Architecture

This project follows a feature-based architecture focused on scalability, maintainability and reusable UI components.

- Feature-based structure
- Shared Components
- Design Tokens
- Theme Engine
- Clean SCSS Architecture
